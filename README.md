# Kai Akihiro / 甲斐 晶大

個人ウェブサイト

このサイトはGitHub PagesとJekyllを使用しています。

## 技術スタック

- **Jekyll**: 静的サイトジェネレーター
- **Alembic Theme**: ベーステーマ
- **GitHub Pages**: ホスティング
- **Noto Sans JP / Noto Serif JP**: 日本語フォント

## カスタマイズ

- 日本語対応
- カスタムカラースキーム（ブルー系グラデーション）
- 日本語フォント最適化
- モダンなUIスタイリング

## ローカル開発

```bash
bundle install
bundle exec jekyll serve
```

サイトは `http://localhost:4000` で閲覧できます。

## ライセンス

このサイトはAlembicテーマをベースにしています。
Alembic theme by [David Darnes](https://github.com/daviddarnes/alembic)
